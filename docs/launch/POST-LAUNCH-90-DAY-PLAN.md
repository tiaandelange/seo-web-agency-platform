# Post-launch 90-day plan

Cadence anchors: 30 min daily (week 1), then weekly working session (~2 h), monthly review (~half day). All findings logged in PROJECT-STATUS.md.

## Days 1–7 — Indexation baseline

- Daily: GSC Page indexing + URL Inspection of home/services/pricing; confirm sitemap "Success"; crawl stats sane.
- Verify the deliberate noindex set is excluded exactly (thank-you, project templates/categories).
- Fix anything Google reports (soft 404s, redirect errors) same-day.
- Record baseline: indexed count, impressions (will be ~0 — that's fine), CWV lab scores.

## Days 8–30 — First signal

- Weekly: Performance report — first queries/impressions per page; note which clusters surface first (they get priority content investment).
- Publish: first phase-2 content batch per the roadmap (remaining compare pages' polish, first new article).
- CTR pass 1: any page with impressions >100 and CTR <1.5% gets title/description rewrite (validator re-run after).
- Citations: GBP live; LinkedIn company page; Clutch/TechBehemoths/GoodFirms listings under the final name.
- Conversion check: forms tested weekly; any real enquiries → response-time discipline (<4 business hours target).

## Days 31–60 — Expansion

- Content: 2–3 roadmap articles (cost-guide cluster first); each interlinked per the map on publish day.
- Internal-linking review: pages with impressions but no clicks get richer contextual links from stronger pages.
- Real evidence: first project case study drafted if any project completed (CASE-STUDY-FRAMEWORK conversion steps).
- Technical: CWV field data check (may still be insufficient traffic — use lab); fix any GSC enhancements errors.
- Query mining: search terms with impressions that no page targets → keyword map additions (or FAQ answers) — never spontaneous new pages without map entry.

## Days 61–90 — Review and double-down

- Monthly review #3: per-cluster scorecard (impressions, avg position, clicks, conversions) vs TOPICAL-AUTHORITY-PLAN sequence; declare cluster 1 status.
- CTR pass 2 on everything with >200 impressions.
- Publish: 2–3 more articles; second case study if available; Johannesburg content deepened if signals justify.
- Conversion review: quote submissions by landing page; kill/fix pages that get traffic but never assist conversions (usually a CTA or content-promise mismatch).
- Decide next quarter: Cape Town page justified? `ecommerce-guides` category opening? Next.js service page split-out (keyword map notes)?

## Standing weekly checklist (continues after day 90)

| Check | Where |
|---|---|
| New queries + CTR outliers | GSC Performance |
| Indexing anomalies | GSC Pages |
| Enhancement errors | GSC Enhancements |
| Form deliverability | Test submit |
| Uptime/SSL | Host dashboard |
| Publication cadence: ≥2 substantive pieces/month, ≥1 case study/quarter | Roadmap CSV |

## Explicit non-goals in the first 90 days

Link-buying, directory spam, AI-bulk content, new location pages without evidence, rank-guarantee promises in any sales conversation (rules 10–12 live on after launch).
