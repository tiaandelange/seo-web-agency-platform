# Google Business Profile checklist

Precondition: final business name chosen and consistent everywhere (LOCAL-SEO-PLAN.md — do NOT create the profile under the working name).

## Setup

- [ ] Create/claim profile at business.google.com with the permanent Google account
- [ ] Business name: exact trading name, nothing appended ("Meridian Web Systems", never "Meridian Web Systems | Web Design Pretoria" — keyword-stuffed names risk suspension)
- [ ] Type: **Service-area business** (no address shown) unless a real staffed office is published (A-03 / input #4)
- [ ] Service areas: Pretoria, Johannesburg, Centurion (+ confirmed additions) — match `config/brand.ts`
- [ ] Primary category: **Website designer**
- [ ] Secondary categories: Software company (systems work), Internet marketing service (only if honestly applicable)
- [ ] Phone: the one canonical number; Email: domain-based address
- [ ] Website URL: production home with `?utm_source=google&utm_medium=organic&utm_campaign=gbp`
- [ ] Hours: match brand config (Mon–Fri 08:00–17:00 placeholder — confirm first)
- [ ] Description (≤750 chars): plain factual summary of services + areas; no promises, no keyword stuffing
- [ ] Services list: mirror the 11 service names with 1–2 sentence descriptions each
- [ ] Attributes: identify-as options as applicable; appointment link → /contact/

## Content assets

- [ ] Logo + cover image (after visual phase; interim: simple wordmark)
- [ ] 3–5 real photos when available (workspace, founder, project screenshots with permission) — no stock
- [ ] First Google Post: launch announcement linking the site

## Review process (from LOCAL-SEO-PLAN.md)

- [ ] Generate the short review link; store in the client-offboarding template
- [ ] Ask at project completion; one reminder max; never incentivised or gated
- [ ] Respond to every review within a week

## Verification + NAP

- [ ] Complete Google's verification (video verification is common for service-area businesses — have registration docs and work evidence ready)
- [ ] NAP identical across: website footer/contact, GBP, LinkedIn, any directory — tracked in a citation sheet
- [ ] Add GBP URL to `config/brand.ts` → `googleBusinessProfile` once live (feeds nothing publicly yet; internal record)

## Metrics (monthly, into the 90-day plan reviews)

Calls, website clicks, direction requests (should be ~0 as service-area), search terms shown in GBP performance.
