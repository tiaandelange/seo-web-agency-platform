# Final name research

Date of all checks: **2026-07-21** (via web search and DNS lookup from the development machine). Scope: the existing shortlist (`NAME-SHORTLIST.md`) plus new candidates generated where the shortlist proved weak.

## Verification honesty rules (binding)

- Every finding below is **provisional**. "No obvious conflict found" is the strongest claim made — it is *not* "available".
- CIPC company-name and trademark registers were **not** queried directly (they require the official CIPC/e-services and IP-online platforms). Only the owner or an attorney can produce authoritative results there.
- Domain findings are DNS lookups only (a domain with no DNS can still be registered and parked without nameservers). Authoritative answer requires a registrar/WHOIS query at registration time.
- Social handles were not individually verified on each network; noted only where a conflict surfaced in search.

## Method

1. Web search per name: `"<name>"` + company/web/software, globally and with South Africa qualifiers.
2. DNS lookup (`Resolve-DnsName`, A and NS records) for the natural `.co.za` and `.com` forms.
3. Pronunciation/spelling/acronym review against the blueprint criteria.
4. Weighted scoring (matrix: `NAME-SCORING-MATRIX.csv`; criteria and weights as specified by the owner brief).
