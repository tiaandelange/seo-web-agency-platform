## Recommendation (preliminary)

1. **Modulus Digital** — engineering-native word that flatters the target audience, scales from websites to software, premium tone.
2. **Datum Web Systems** — quiet engineering reference + explicit “systems” breadth.
3. **Trueline Digital** — plainest to say over the phone; accuracy connotation matches the brand promise.

Wildcard: **Bronberg Digital** if a strongly Pretoria-rooted identity is wanted.

“Meridian Web Systems” remains the **temporary working brand** in `config/brand.ts` until a final choice is made; it deliberately scores mid-table so nobody gets attached to it.

## Notes on SEO usefulness

No shortlisted name contains “website design” keywords — deliberate. Exact-match names skew low-cost (“Pretoria Web Design”) and cap expansion. Rankings come from page architecture, not the brand string; the brand’s SEO job is to be memorable and to attract branded searches later.

## Verification checklist before committing (all currently unchecked)

1. `.co.za` and `.com` availability at a registrar (mark date checked).
2. CIPC company-name search + reservation.
3. SA trademark search (CIPC IP office) + TMView for international conflicts, class 42 (and 35).
4. Google/LinkedIn search for confusable trading names, especially SA agencies.
5. Social handle availability (LinkedIn company, X/Instagram optional).
6. Say-it-over-the-phone test with five strangers.
